Inisialisasi Project Angular dengan Angular-CLI

Angular-CLI merupakan skeleton generator untuk project Angular, dengan bantuan Angular-CLI kita tidak perlu lagi setup banyak kebutuhan dasar seperti susuna folder, cara run, cara build dan lain-lain. Kesemuanya itu telah dibuatkan perintah (command) melalui Angular-CLI. Seperti halnya Ember.js yang mempunyai Ember-CLI, Vuejs dengan Vue-CLI dan React dengan create-react-app maka seperti itu pula fungsi Angular-CLI bagi Angular.

Pada saat artikel ini ditulis Angular-CLI sudah mencapai versi stable yakni 1.0.4, Anda bisa lihat release history lebih lengkapnya disini. Dengan sudah mencapai versi stable artinya Angular-CLI juga sudah siap untuk dipakai di production atau project dengan skala yang lebih besar dan kompleks. Secara fitur sudah dikatakan mumpuni dan memenuhi kebutuhan para developer namun juga akan tetep ada perbaikan dan penyesuaian secara berkala yang biasanya tidak akan breaking versi yang lebih lama.

Berikut adalah perintah sederhana untuk inisialisasi project Angular menggunakan Angular-CLI :

  1. Pastikan kamu sudah memasang nodejs ya, kalau belum bisa download disini
  2. Pasang Angular-CLi di global dengan cara :
npm install -g @angular/cli

3. Buat project baru dengan cara :

ng new PROJECT_NAME

4. Setelah menjalankan command ini, Angular-CLI akan men-generate folder struktur yang dibutuhkan oleh kita, juga meng-install semua depedency yang dibutuhkan. Proses ini akan memakan waktu yang lumayan lama tergantung dari kondisi internet Anda.

5. Setelah semua proses di no.4 selesai, kita bisa menjalankan project pertama kita. Pastikan bahwa Anda berada di dalam folder project ketika menjalankan project.

// untuk pidah ke folder project
cd PROJECT_NAME
// untuk menjalankan project
ng serve

6. Dan pekerjaan kita sudah selesai. 
Kita bisa membuka project ini di alamat http://localhost:4200/, dan akan melakukan hot reload ketika terjadi perubahan dalam file-file project kita.

Selain untuk meng-inisialisai project awal, Angular-CLI juga bisa digunakan untuk membuat berbagai macam file di Angular seperti component, directive, pipe, service, class, guard, enum dan module.
