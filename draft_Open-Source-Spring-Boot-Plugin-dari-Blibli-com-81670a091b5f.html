Open Source Spring Boot Plugin dari Blibli.com

Menceritakan beberapa Spring Boot Plugin yang pada perjalannya kita open source kan untuk teman-teman.

Seperti sudah banyak diketahui oleh teman-teman di luar bahwa kami di Blibli.com banyak sekali menggunakan teknologi yang berbasiskan JVM terutama Spring besutan dari Pivotal mulai dari SpringMVC, Spring Boot, Spring Cloud dan banyak lagi lainnya. Pada tulisan kali ini kita akan menceritakan bagaimana Blibli.com memberikan kontribusi balik terhadap komunitas dengan membuka beberapa kode ke open-source.

Berikut beberapa Spring Boot Plugin yang telah di release oleh Blibli.com sejauh ini sampai artikel ini dipublikasikan:

➡️ command-plugin

Blibli.com pada banyak kode terbaru menerapkan salah satu design pattern yakni Command Pattern. Plugin ini banyak kita gunakan untuk implementasi design pattern ini.

➡️ swagger-plugin

Menggunakan plugin di Spring Boot kini cukup menambahkan decorator @Api di dalam class controller kalian.

➡️ template-plugin

Plugin ini membantu kita untuk me-render template view menggunakan mustache/handlebars.

➡️ kafka-plugin

Plugin ini akan membantu kita dalam menggunakan Apache Kafka di dalam Spring Boot.

➡️ common-plugin

Plugin ini berisi beberapa common helper yang akan membantu dalam mengembangkan Api untuk menggunakan Spring Boot. Kita bisa mendapatkan helper seperti paging, error handler, validation, serta beberapa helper untuk membuat API response.

Untuk daftar selengkapnya Anda bisa langsung mengunjungi Github Blibli.com atau langsung mengunjungi tautan berikut untuk data terbaru yang berkaitan dengan Spring Boot : https://github.com/search?q=topic%3Aspring-boot+org%3Abliblidotcom+fork%3Afalse
