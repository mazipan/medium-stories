So, what is it React-Simplert ?

React-Simplert is my porting component from Vue.js that I developed before into React technology. React-Simplert is a simple alert component like SweatAlert that written in React and can be reuse by your project with ease. All the styling is just ported from the previous one, so will be very same with Vue.js version. Feature that included hopefully will be same too with Vue.js version but maybe will be there is some little different in implementation.

In Vue.js version, I suggest the Users just call the inner method with $refs to doing modification like showing and closing Alert. But in React, using refs and call the method directly was not recommendation trick. React suggest to pass the changes via Props, so I pass all the setup for this React-Simplert via Props too.
